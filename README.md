# Celular Store

Site de venda de celulares e eletrônicos. Apenas moderadores (definidos por IP) podem cadastrar produtos. Usuários comuns podem comprar.

## Estrutura
- Backend: Node.js + Express + SQLite
- Frontend: React

## Funcionalidades
- Cadastro de produtos (moderador)
- Listagem e compra de produtos (usuário)
- Moderação por IP

## Como rodar
Veja instruções nas pastas `backend` e `frontend`.
